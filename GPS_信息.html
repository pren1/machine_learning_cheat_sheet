名词解释

当然可以。下面是你所列出名词的详细解释：

  1. GPS和GNSS定位引擎开发:

    • GPS: 全球定位系统(Global Positioning System)，是美国建立的一个全球卫星导航系统。

    • GNSS: 全球卫星导航系统(Global Navigation Satellite System)是一个通用的术语，指的是所有的卫星导航系统，包括GPS(美国)、GLONASS(俄罗斯)、BeiDou(中国)和Galileo(欧洲)等。

  2. GNSS测量和误差来源: 如前所述，误差来源可能包括电离层和对流层延迟、多径效应、接收机噪声、卫星轨道误差、卫星钟差和几何稀疏性等。

  3. 估算技术:

    • 最小二乘法(Least Squares): 是一种数学优化技术，用于最小化预测值与实际观测值之间的总平方误差。在GNSS中，它用于确定接收机的位置。

    • 卡尔曼滤波(Kalman Filtering): 是一种迭代估算算法，用于预测系统的状态。它在面对噪声和不确定性时表现良好，常用于GNSS定位，尤其是在信号受到干扰或遮挡时。

  4. 错误/故障检测与排除(Blunder/Fault Detection and Exclusion):

    • 这是GNSS系统中的一个过程，用于检测并排除可能导致定位错误的异常测量。

  5. 多星座(Multi-Constellation)与多频率(Multi-Frequency):

    • 多星座: 使用来自多个卫星导航系统（如GPS, GLONASS, BeiDou, Galileo）的信号进行定位。

    • 多频率: 使用卫星信号的多个频率进行定位。多频率可以帮助纠正某些误差，例如电离层引起的误差。

  6. AGNSS:

    • 辅助全球导航卫星系统(Assisted GNSS)。它使用外部信息来辅助GNSS接收机，从而加速初始定位并提高定位的准确性。例如，一个移动电话可能会使用基站的数据来辅助其内部的GNSS接收机。

GPS误差来源

当然了解，博士研究往往是非常深入并且专注于某个特定的领域。对于GPS和GNSS的知识，我可以为你提供一个简单的概述，这样你可以有一个基本的了解，并在面试时更有信心。

GPS和GNSS误差来源及概念：

  1. 大气延迟:

    • 电离层延迟: 电离层中的自由电子会导致GPS信号的延迟。这种延迟可以通过使用双频GPS接收机来部分地校正。

    • 对流层延迟: 信号通过对流层时会受到湿度和温度的影响。一般来说，这种延迟是比较难以预测的，但在高精度的应用中会有特定的模型来进行校正。

  2. 多径效应: 当GPS信号在到达接收机之前反射或折射多次时，会产生误差。这种情况常常在建筑物或大型物体附近发生。

  3. 接收机噪声: 由接收机硬件引起的误差，比如热噪声。

  4. 卫星轨道误差: 卫星的实际位置与广播的星历数据之间的微小差异。

  5. 卫星钟差: 卫星上的原子钟与GPS时间之间的微小偏差。

  6. 几何稀疏性: 如果卫星的位置分布不均或只有很少的卫星可见，可能会影响位置的精确度。

对于估算技术：

  • 最小二乘法: 是一种估算未知参数的数学方法，经常用于GPS解算中。

  • 卡尔曼滤波: 是一种迭代的算法，用于估算系统的状态，并在面对噪声和不确定性时优化其性能。

这些误差都是怎么解决的？

许多这些误差来源在接收终端或用户层面是不能从根本上消除的。但是，我们可以采取一些措施来减轻或纠正这些误差的影响。

  1. 电离层和对流层延迟:

    • 虽然我们不能控制天气或电离层的变化，但可以通过使用多频GNSS接收机来进行误差补偿。不同频率的信号在电离层中有不同的传播速度，所以通过比较这些信号，可以估算出电离层引起的误差并进行纠正。

    • 对于对流层延迟，常用的方法是利用地面站的气象数据和经验模型来估计误差。

  2. 多径效应:

    • 可以通过使用具有高抗多径性能的天线来减少多径效应。

    • 还可以使用多径估计算法和技术来检测并减少其影响。

  3. 接收机噪声:

    • 优化硬件设计，例如使用低噪声放大器。

    • 通过软件算法对信号进行处理，以降低噪声的影响。

  4. 卫星轨道误差:

    • 通过使用精密轨道数据或卫星星历来纠正。这些数据通常由专业组织发布，并提供了卫星的精确轨迹。

  5. 卫星钟差:

    • 同样可以使用卫星星历中的数据来进行纠正。

  6. 几何稀疏性(Geometric Dilution of Precision, GDOP):

    • 选用更佳的卫星组合。当你的接收机可以“看到”更多的卫星，特别是分布在天空不同位置的卫星时，GDOP会更低。

    • 使用其他GNSS系统的信号（如GLONASS、BeiDou、Galileo）可以增加可用的卫星数量。

虽然很多误差不能从源头上消除，但通过上述措施，我们可以有效地减少这些误差对定位精度的影响。

Blunder/Fault Detection and Exclusion 这又是啥，详细说说

“Blunder/Fault Detection and Exclusion”（BFD&E）通常是在卫星导航系统（如GPS和GNSS）中使用的一个术语。在这种环境下，确保数据的正确性和可靠性至关重要。因为一个或多个不准确的卫星测量可以严重地影响定位的准确性。BFD&E涉及的是检测和/或纠正这些错误。

  1. Blunder/Fault Detection（错误/故障检测）:

    • 这是一个过程，用于识别异常或错误的测量值。通常基于统计检验来完成，比如χ^2检验。

    • 当系统收到一个或多个卫星信号时，它会预期得到一个特定范围内的测量值。如果测量值偏离这个预期范围太多，系统就会认为这是一个“blunder”或“fault”。

  2. Exclusion（排除）:

    • 一旦检测到异常或错误的测量值，系统可能会选择忽略或“排除”这些值，以确保它们不会影响定位结果。

    • 这通常涉及重新计算位置，不使用被认为是有问题的测量值。

为什么这很重要？考虑一下飞机或其他关键基础设施使用GPS来导航。一个不准确的测量值可能会导致飞机偏离预定的航路，这可能是灾难性的。因此，GNSS接收机具有检测和纠正这些问题的能力是至关重要的。

具体到GNSS的应用中，常见的方法包括Raim（接收机自主完整性监测），它使用统计方法来检测和排除错误的卫星测量值。

总的来说，Blunder/Fault Detection and Exclusion是GNSS系统确保其测量和定位准确性的关键组成部分。

χ^2检验（Chi-square test）是一种统计检验，用于确定观察到的频数与期望频数之间的差异是否有统计意义。它主要用于分类数据。

为了给你一个简单的理解，让我们考虑一个基本的例子：

假设你有一个骰子，你想知道它是否公平。你投掷骰子600次，记录下每一面的出现次数。如果骰子是公平的，你期望每一面平均出现100次。但你观察到的频数可能与此略有不同。

为了确定观察到的频数与期望频数之间的差异是否是由于随机变异（即骰子是公平的）还是骰子是有偏的，你可以使用χ^2检验。

χ^2统计量的计算公式是： χ2=∑(O−E)2E 其中 O 是观察到的频数，E 是期望的频数。

然后，你会与一个χ^2分布的关键值进行比较（根据自由度和所选的显著性水平来确定）以决定差异是否有统计意义。

在实际应用中，许多统计软件包（如Python的SciPy）都提供了进行χ^2检验的功能。

尽管χ^2检验的背后有深入的统计知识，但如果你只需要对其有基本的了解，理解上述内容应该就足够了。然而，如果你希望能够在实际情境中应用它，确实需要花些时间深入学习和练习。

当然可以。最小二乘法是一种估计参数的方法，目标是使得模型预测的输出与实际观察数据之间的差的平方和达到最小。

在GNSS定位中，我们有以下基本方程式： ρi=(x−xi)2+(y−yi)2+(z−zi)2+b+ei

其中：

  • ρi 是从第 (i) 颗卫星到接收机的观测距离（伪距）。

  • (x,y,z) 是接收机的位置坐标。

  • (xi,yi,zi) 是第 (i) 颗卫星的位置坐标。

  • b 是接收机钟差。

  • ei 是误差项，包括各种误差如多径效应、大气延迟等。

为了确定接收机的位置，我们需要解这个方程组。但是，由于各种误差，观测的伪距ρi并不完全准确。因此，直接求解可能不会得到准确的结果。

最小二乘法为我们提供了一个解决方案。我们不是直接求解方程组，而是尝试找到一个接收机的位置 (x,y,z) 和钟差 b，使得所有观测误差 ei 的平方和达到最小。

换句话说，我们尝试最小化以下函数： J=∑i(ρi−(x−xi)2+(y−yi)2+(z−zi)2−b)2

最小化这个函数，我们就可以得到一个最佳估计的接收机位置和钟差，即使在存在误差的情况下。

这个方法在有噪声或其他误差存在的数据中经常被使用，因为它提供了一个鲁棒的方法来估计模型参数。

最小二乘法的核心思想是找到参数值，使得模型预测值与实际观测值之间的差异平方和最小。

假设我们有一个线性模型 y=Ax+b 其中

  • y 是观测值向量

  • A 是已知的设计矩阵

  • x 是参数向量

  • b 是误差向量（通常假设为高斯噪声）。

我们希望找到 x，使得 bTb 最小。

将模型代入，我们得到： y=Ax+b ⇒b=y−Ax

我们希望最小化 bTb，即： minx(y−Ax)T(y−Ax)

对上式进行展开，我们得到： minx(yTy−2xTATy+xTATAx)

为了找到最小化上式的 x，我们可以对 x 求导并令其为零。将导数设为0，我们得到正规方程组（Normal Equations）： ATAx=ATy

如果 ATA 是非奇异的（即可逆的），那么该方程组的解为： x=(ATA)−1ATy

这就是最小二乘的线性解。在实际应用中，可以使用多种数值方法来求解这个方程组，例如高斯消元、LU分解、QR分解等。

需要注意的是，这里描述的是简单的线性最小二乘。在非线性情况下，求解会更为复杂，通常需要迭代方法，如牛顿法或梯度下降法等。

伪距平滑和载波相位测量涉及到卫星导航系统（如GPS）的核心测量技术。为了理解它们，我们首先需要了解两种主要的GNSS测量方法：伪距测量和载波相位测量。

  1. 伪距测量：

    • 伪距测量是最常见的GNSS测量方法。

    • 它基于卫星发射的信号与接收机接收到的信号之间的时间差来计算距离。

    • 由于各种误差源（例如大气延迟、接收机钟差等），测得的距离被称为“伪距”。

    • 伪距测量受到多径效应的影响，这是由于信号经由多个路径到达接收机引起的。

  2. 载波相位测量：

    • 载波相位测量是基于接收的信号的相位来进行的。

    • 相位测量比伪距测量更为精确，但也更容易受到短期的小波动或突变的影响。

    • 由于载波相位测量对多径效应更为敏感，它可以被用来识别和纠正伪距测量中的多径误差。

伪距平滑是一个技术，它结合了伪距测量和载波相位测量的优点。具体来说，它使用载波相位测量来修正和平滑伪距测量值。因为载波相位测量更精确（但更敏感），所以通过将其与伪距测量相结合，我们可以得到一个既稳定又精确的位置估计。

这种技术在实践中非常有用，尤其是在多径效应较为严重的环境中，如城市峡谷或森林。

具体怎么做呢？

伪距平滑通过结合伪距测量和载波相位测量来改善位置估计的准确性。这里，我们将更深入地探讨这个过程的核心概念和操作：

  1. 初步的位置估计：首先，使用伪距测量从多个卫星来获取一个初步的位置估计。

  2. 计算伪距和载波相位之间的差异：这一差异通常被称为“载波-伪距距离差”或“载波-伪距平滑误差”。在没有任何干扰的理想条件下，这两种测量方法应该会给出相同的距离。但由于各种误差（尤其是多径效应），它们往往会有所不同。

  3. 伪距平滑：为了平滑伪距测量，可以使用一个简单的滤波器，比如一阶或二阶卡尔曼滤波器。这个滤波器会结合载波-伪距距离差和之前的伪距测量来更新当前的伪距测量。这样，你就获得了一个经过载波相位校正的平滑伪距。

  4. 更新的位置估计：使用经过平滑的伪距测量从新得到一个位置估计。

  5. 迭代和持续更新：当新的伪距和载波相位测量值可用时，这个过程可以持续地迭代和更新。

在伪距平滑过程中，载波相位测量值提供了一个参考或基准，可以帮助纠正伪距测量中的短期波动或误差。而伪距测量由于其本身的稳定性（尽管可能不太准确）则为整个过程提供了一个稳固的基础。这两种测量方法的结合提供了一个既精确又稳定的位置估计，特别是在受到多径效应干扰的环境中。

请注意，实际操作可能会根据具体的应用和环境进行调整。但上述流程提供了一个基本的框架，描述了伪距平滑的核心思想和步骤。

载波相位比伪距更精确，我可以这样理解吗？

是的，您可以这样理解。载波相位测量是基于GNSS信号的相位变化进行的，而这种测量通常具有非常高的精度。具体来说，载波相位可以达到分米甚至毫米级别的精度。

相比之下，伪距测量是基于GNSS信号的传播时间进行的，受到各种误差源的影响（如电离层和对流层延迟、卫星钟差、多径效应等），其精度通常在米级别。

然而，载波相位测量也有一个主要的缺点，那就是“整周模糊度”问题。由于相位测量实际上只能给出相对的相位差异，因此在连续的信号接收中，可能会出现不确定的整数周数的差异。这种不确定性使得单独使用载波相位测量来确定绝对距离变得复杂。

但是，当我们结合伪距测量时，可以利用其提供的粗略距离估计来解决载波相位的整周模糊度问题，并进一步提高位置的精确度。这就是为什么在实际的GNSS应用中，伪距和载波相位测量通常会结合使用。

相位的值是不是一个角度？0～360？

不完全是。载波相位确实表示为一个角度，但它不仅仅限于0-360度的范围。相位测量值是一个连续的角度变化，它可以超过360度。这是因为当GNSS信号从卫星传播到接收机时，载波波形可能经历多次完整的循环（每次循环对应360度）。

例如，考虑一个卫星到接收机的距离变化，使得载波波形经过了5.5个完整的循环。相位测量值将是1980度（5个循环为1800度，再加上0.5个循环的180度）。

但在处理相位测量时，通常对其进行模360操作以获取其当前循环内的相对角度。然而，连续的总循环数（即整周数）是位置解算中一个关键的未知数，需要通过其他技术来确定。

所以，当我们说“相位”时，它确实表示一个角度，但这个角度可以是任何值，并不仅仅限于0-360度范围。

我差不多都理解了，但是还有一个不懂。载波相位怎么就能缓解多径效应？

多径效应是由于GNSS信号在到达接收机之前被反射、折射或散射引起的。这些反射的信号与直接的卫星信号混合，导致接收机测量到的信号延迟，进而影响定位精度。

载波相位和伪距测量在处理多径效应时有不同的特点：

  1. 频率与多径效应：载波的频率比伪距测量（基于代码）的频率要高。高频信号对多径效应的敏感性较低，因为它们的波长较短，所以反射的信号与直接信号之间的相位差异更容易区分。

  2. 相位的连续性：载波相位测量非常连续，而伪距测量更容易受到突然的跳变。多径效应通常引入相对较小的相位错误，这些错误在连续的载波相位测量中更容易被平滑处理。

  3. 误差的性质：在伪距测量中，多径效应引入的误差与信号的幅度变化有关，而在载波相位测量中，误差与信号的相位变化有关。由于载波相位的连续性和高频特性，它通常更能有效地处理多径效应引入的相位误差。

  4. 组合使用伪距和载波相位：一种常用的策略是利用载波相位测量对伪距测量进行平滑，从而降低多径效应和噪声的影响。这种“平滑”策略结合了伪距测量的初步距离信息和载波相位测量的高精度、低噪声特性。

总的来说，载波相位由于其连续性和高频特性，比伪距更能抵抗多径效应的影响。但是，为了最大限度地提高定位精度，通常会结合使用伪距和载波相位测量。
